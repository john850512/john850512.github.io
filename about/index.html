關於我

HI，我是John，一個在台灣就讀資訊工程，碩士畢業正在工作的資深工程師。研究領域包含資料科學、機器學習和深度學習。工作領域包含軟韌驅動開發、IP硬體驗證。

很久以前無名小站還在的時候就很喜歡寫網誌，因為覺得跟別人分享我最近都在幹什麼是一件挺有趣的事情，後來無名倒了就只剩下偶爾在臉書上發發文章。但因為FaceBook的內容又是無法被搜尋到的，很多珍貴的內容都有不見的風險，又或者找個以前的文章要一直往上滑滑到手快酸死還是找不到…所以我決定設立一個網站將一些文章放上來保存。

跟網路上的許多同年齡的大神比起來，我的程式能力非常的弱，所以我正在學習中，儘管腳步很慢，但我希望自己也能向他們看齊。我很享受這段過程，也因此希望能把這段過程記錄下來，希望將來回顧時能有滿滿的文章可以欣賞。

Anyway，在這裡我會分享一些程式題目的解法、或是我去參加了什麼東西並學到了什麼啦、或是我最近生活上遇到什麼障礙啦，也有可能是我去哪兒玩呀、遇到什麼新奇事之類的~~

你知道，如果一天到晚只是Coding，那真的是件很無聊的事，對吧?

Blog經營歷程

下面簡單敘述一下一路上的經營網誌歷程~
現在的Blog架在: Github page，使用了Hexo作為靜態網站的開發框架

無名小站時期(2008~2013)

從2008年，也就是國中時期開始接觸寫網誌，那時候主要是寫一堆日常生活發生的事情，以及遊戲的小小創作(楓之谷的小漫畫xD)

當初在經營無名小站的時候經營得很開心，努力上網找了各式各樣的資源，學習如何修改樣式、播放音樂、更改鼠標…

還有如何寫一個增加流量的外掛(當時剛接觸程式設計，然後看到網路上有類似的工具覺得很有趣就開始研究)，每天都刷個幾千次的瀏覽次數，然後閱覽數就這樣突破幾十萬了覺得很有成就感xD

不過現在想想當初發的文真的是有點屁…不忍回顧當初到底寫了什麼…

痞客邦時期(2013~2016)

後來很可惜的是，無名小站倒了，所以我將無名小站的資源全部備份到痞客邦上。

不過由於痞客邦用起來一直沒有很順手，所以我也沒有花太多心思在經營跟寫作上，高中三年幾乎沒有在寫部落格。

Wordpress.com時期（2016~2020)

後來大學讀了資訊工程學系，由於在自主學習上想要有一個可以記錄的平台，像是學習筆記這樣。所以又重新開始尋找適合我當時需求的平台。

後來找到了Wordpress，Wordpress提供了很方便的後台以及還不錯用的文章編輯器，所以就決定在這上面重新開始寫文章。

順便說一下，Wordpress有分Wordpress com和Wordpress org

  • com是他們也幫你代管了你的網站，並且有免費版可以讓你用，不過免費版不能安裝額外的外掛插件以及只能使用很少的template
  • org是你要自己找一個主機來架server，不過這種的自由度相對大很多，想裝什麼插件就裝什麼、想改啥就改啥

不過我當時不想要每個月花錢找主機商幫我架網站，就選了wordpress com…這也導致了後續的悲劇QQ

Github page時期(2020~)

儘管Wordpress.com的內建編輯器支援數學公式跟程式碼，不過實在是對於有這些需求的人很不友善。每次辛辛苦苦打完的程式碼送出後總是會被轉換成些多餘的符號字串，然後就必須要再度編輯把這些改回來，例如下面這張圖。

而對於數學公式則是要加上一些”$ latex $“以及一些後綴參數才能去控制符號的顏色跟字體大小，如果有用過Markdown的人就知道這些對於撰寫者來說有多不方便，我明明只想專注寫作阿

(☍﹏⁰)

儘管上述的諸多不便，不過我還是持續在Wordpress這個平台上經營了四年。不過在前些日子，因為一些機會剛好在研究如何用Hexo在Github page上架設靜態的網站，幫別人架好他的網站後我就一直思考，自己是不是應該也把平台轉過去，讓未來的自己能更方便的寫作文章。前面講到wordpress的大問題在Hexo框架都能很輕易被解決，Hexo可以讓你無縫接軌Markdown，讚讚。

後來決定好了之後就經過了一個月的時間，參考了一個模板開始來設計自己的新部落格，然後不斷地去修改相關的source code做出自己想要的樣子，最後也把wordpress所有文章全數轉過來了。甚至，還下定決心買了一個自己的domain name。(為此每個月將會少喝一杯多多綠，歡迎點擊下方圖片贊助xD)

不過轉來後由於是新的網站，所以以前的流量全部都要砍掉重練(由於我當初適用wordpress com，免費版也不能用外掛301把流量導過來…我又不因為這個花錢)

這真的是太痛心了，之前經營了四年才好不容易累積的六萬多觀看次數就這樣再見了…不過長痛不如短痛…畢竟我不想以後每寫一次文章就要罵一次文章編輯器…
